# Customer Churn Prediction

## Project Overview

This project aims to predict customer churn using machine learning techniques on the Telco Customer Churn dataset.

The objective is to identify customers who are likely to leave a telecommunications service and provide insights that can help improve customer retention strategies.

## Dataset

Telco Customer Churn Dataset

## Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn
* LightGBM
* Matplotlib
* Seaborn
